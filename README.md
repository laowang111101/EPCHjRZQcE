# 图书管理系统 python1217

## 项目概述

本系统是基于 Python 语言，采用 Django 框架以及 SQLite 数据库开发的图书管理系统。系统旨在提供便捷的图书管理、借阅流程以及用户个人管理功能。

## 技术栈

- **后端语言与框架**：Python 3.x，Django
- **数据库**：SQLite
- **前端技术**：HTML/CSS/JavaScript

## 功能模块

### 基础功能

- **登录功能**
  - 用户登录与登出
  - 用户身份验证

### 主要模块

- **图书管理**
  - 图书信息录入与编辑
  - 图书检索
  - 图书分类管理

- **借阅管理**
  - 借阅记录登记
  - 归还处理
  - 超期提醒

- **个人管理**
  - 个人信息编辑
  - 借阅历史查询

- **会员管理**
  - 会员资格管理
  - 会员借阅特权设置

- **数据可视化图表**
  - 借阅数据统计
  - 图书分类统计

## 系统角色

- 系统管理员
- 普通用户
- 会员用户

## 运行环境

- Python 3.x
- Django 2.x
- SQLite 3.x

## 部署步骤

1. 环境配置：安装 Python 3.x，pip，以及通过 pip 安装 Django 和 SQLite 相关依赖。
2. 数据库初始化：执行数据库迁移命令以创建和更新数据库表结构。
3. 系统配置：配置应用设置文件，如静态文件路径、数据库连接等。
4. 运行服务：使用 Django 自带的开发服务器运行应用。
